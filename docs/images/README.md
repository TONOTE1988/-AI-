# デモ画像プレースホルダー

このフォルダには以下の画像を配置予定：

- demo-main.png: メイン画面のスクリーンショット
- demo-chat.png: チャット画面のスクリーンショット  
- demo-init.png: 初期化画面のスクリーンショット

## 画像追加手順

1. Streamlitアプリを起動: `streamlit run app.py`
2. 各画面のスクリーンショットを撮影
3. この docs/images/ フォルダに配置
4. README.mdの画像パスを確認

## 一時的な解決策

現在は画像の代わりに機能説明をテキストで記載しています。
実際の画面確認は以下のコマンドで可能：

```bash
streamlit run demo.py  # デモモード
streamlit run app.py   # 完全版
```
